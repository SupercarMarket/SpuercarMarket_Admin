│  │  └─MemberInfoTable                     : 판매자 정보 -> 회원 정보 테이블
│  ├─SubTitleTable                          : 부제목 테이블
│  └─VehicleDetailTable                     : 차량 설명글
├─ForSaleList
│  ├─ForSaleMainTable
│  └─PageItems
│      ├─Table

├─VehicleRegistrationInquiry                : 판매차량 등록 문의 리스트

│    └─TopBanner                            : Top banner, dropdown, search bar 모음
└─VehicleRegistrationInquiryDetail          : 판매차량 등록 문의 리스트 상세 보기
